# MSCS 634 – Advanced Data Mining Project
**Final Deliverable 4: Final Insights, Recommendations, and Presentation**  
**Date:** April 18, 2026  

## Project Overview
This repository contains the complete undergraduate data mining project using the **Stroke Prediction Dataset** (Kaggle). The project follows the full data-mining lifecycle across four deliverables: data cleaning & EDA, regression modeling, classification, clustering, and association rule mining.

## Repository Contents
- `project-deliverable-4-advanced-data-mining.ipynb` – Consolidated Jupyter Notebook with all code, comments, and visualizations  
- `Deliverable4_Report.pdf` (or .docx) – Full academic report
- `Presentation_Slides.pptx` – PowerPoint slides used in the video  

## Key Results
- Best model: Tuned Decision Tree (Accuracy 92.27%, F1 0.1684)  
- Optimal clusters: 4 (highest-risk cluster stroke rate = 13.4%)  
- Strongest predictors: age, hypertension, avg_glucose_level  

## How to Run
1. Open the notebook in Jupyter or Kaggle  
2. All required libraries are listed at the top  
3. Data is loaded from the Kaggle input path
